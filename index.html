Was ist Html

Die Hypertext Markup Language (HTML, englisch für Hypertext-Auszeichnungssprache) ist eine textbasierte Auszeichnungssprache zur Strukturierung elektronischer Dokumente wie Texte mit Hyperlinks, Bildern und anderen Inhalten. HTML wurde für das World Wide Web als Auszeichnungssprache für Webseiten entwickelt.

Geschichte

HTML (HyperText Markup Language) entstand Anfang der 1990er Jahre. Der britische Informatiker Tim Berners-Lee entwickelte HTML am CERN, um wissenschaftliche Dokumente einfach über das Internet teilen und miteinander verlinken zu können. Die erste Version war sehr simpel und bestand hauptsächlich aus Text, Überschriften und Links. Mit dem schnellen Wachstum des World Wide Web in den 1990ern wurde HTML ständig erweitert. Es kamen Bilder, Tabellen und Formulare hinzu. Verschiedene Browser interpretierten HTML jedoch unterschiedlich, was zu Chaos führte. Deshalb übernahm das W3C (World Wide Web Consortium) die Standardisierung. In den 2000er Jahren stagnierte HTML etwas, während CSS und JavaScript wichtiger wurden. Ab 2008 begann die Entwicklung von HTML5, das moderne Web-Anforderungen besser unterstützte, etwa Videos, Audio, Canvas, semantische Elemente und bessere Formulare – ohne zusätzliche Plugins. Heute ist HTML die Grundlage jeder Webseite. Es strukturiert Inhalte, während CSS für das Design und JavaScript für Interaktivität zuständig sind. HTML wird weiterhin gepflegt und angepasst, bleibt aber bewusst einfach und universell.
